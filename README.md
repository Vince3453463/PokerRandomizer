# PokerRandomizer
Little Python script to do low frequency plays in Poker
1) open script
2) type in desired frequency
3) script will generate a random number and tell you to do the play or not, depending if the number is lower or higher then the number you put in
4) "end" will terminate the script
5) anything else will tell you the repeat the input
